# Cv Bam Bam — AI-Powered CV Screening for Small Businesses

**Live at [cvbambam.com](https://cvbambam.com)**

Upload CVs, describe your ideal candidate, and get a ranked shortlist in seconds. Built for small businesses that don't have an HR department.

## Features

- **Semantic CV matching** — understands meaning, not just keywords
- **Ranked shortlists** — best candidates scored and sorted automatically
- **Talent pool** — save good candidates for future roles
- **CSV export** — download candidate data as a spreadsheet
- **Free to start** — screen your first 10 CVs at no cost

## How It Works

1. Upload a batch of CVs (PDF or DOCX)
2. Describe the role or your ideal candidate
3. Get a ranked list with match scores
4. Shortlist, export, or save to your talent pool

## Tech Stack

- **Backend:** Python, FastAPI, SQLite
- **AI:** Hugging Face Inference API (BGE embeddings for semantic matching)
- **Frontend:** Tailwind CSS, vanilla JavaScript
- **Hosting:** Render
